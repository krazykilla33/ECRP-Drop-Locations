# Drop Location Map

A standalone Leaflet map with:

- San Andreas and Cayo Perico maps
- Satellite and atlas views
- Searchable location list
- Click-to-focus markers
- Built-in marker placement mode
- Browser autosave through localStorage
- Copy or download updated `drop-locations.js`

## Use it

1. Upload this entire folder structure to the root of your GitHub Pages repository.
2. Open `drop-locations.html` through GitHub Pages.
3. Press **Edit**.
4. Select a location from the sidebar.
5. Click the exact point on the map.
6. Repeat for every location.
7. Press **Download JS**.
8. Replace `data/drop-locations.js` in your repository with the downloaded file.
9. Commit the change. The markers are now permanent for everyone.

Coordinates are also automatically saved in your current browser while you work, so refreshing the page will not erase your progress.
